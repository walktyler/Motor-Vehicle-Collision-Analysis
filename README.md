# Motor-Vehicle-Collision-Analysis
An Analysis on New York City Motor Vehicle Collisions, what causes traffic accidents, who are most impacted, and suggestions for revisions.

# Project Background
The Motor Vehicle Collisions crash dataset contains details on crash events in New York City.
Each row represents a crash event. The data is sourced from all police reported motor vehicle collisions in NYC.

This dataset has significant amounts of data on traffic accidents, injuries and fatalities. This project thoroughly analyzes this data in order to uncover critical insights that will highlight Traffic-related accidents across New York City.
Insights and recommendations are provided on the following key areas:
- Fatalities & Severity
- Time & Risk
- Road User Vulnerability
- Geography & Equity
